# Network-failover-strategy
This is a network failover strategy implemented in CISCO Packet tracer. When the wifi connection goes down our devices switch to LTE connection. Similarly this project mimics the connection to internet through a FIBER_ISP and SERIAL_LTE DATA connections, when the FIBER_ISP connection goes down the router automatically switches to the LTE connection.
Using Static Routing the routers can communicate among themselves and the branch can switch the networks from FIBER_ISP to LTE connection automatically according to the preference. (High preference of the FIBER_ISP).
